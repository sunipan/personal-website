# My Personal Website

A modern portfolio website built with the latest web technologies and deployed on Vercel.

## Tech Stack

- **Next.js 16.1.6** - App Router for optimal performance
- **React 19.2.4** - Latest React features
- **TypeScript 5.9.3** - Type-safe development
- **Tailwind CSS 4.1.18** - CSS-first configuration
- **GSAP 3.14.2** - Scroll animations with ScrollTrigger
- **Vercel Analytics** - Performance monitoring

## Requirements

- Node.js 20.9 or higher

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start
```

## Animations

The website features smooth scrolling animations powered by GSAP (GreenSock Animation Platform) with the ScrollTrigger plugin. Key animations include:

- Hero section entrance animations
- 3D flip card for skills showcase
- Pinned scroll sections for experience timeline
- Smooth transitions throughout

## Deployment

Deployed on Vercel with automatic deployments from the main branch.
